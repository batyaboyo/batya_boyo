# Batya Boyo - Cybersecurity Portfolio

A professional, high-performance portfolio website for a Cybersecurity Specialist and Full-Stack Web Developer. Built with a "Hacker/Terminal" aesthetic, this site showcases technical expertise in SOC Analysis, DFIR, Red Teaming, and AI Security.

## 🚀 Features

- **Terminal Aesthetic**: A custom, interactive CLI-inspired UI with a dedicated terminal section.
- **Interactive Terminal**: Functional CLI with commands like `whoami`, `ls`, `projects`, `skills`, and more.
- **Dynamic Projects**: Filterable project grid showcasing work in Cybersecurity, Development, and IT.
- **Theme Toggle**: Support for both Dark (Matrix/Terminal) and Light (Brutalist) hacker themes.
- **Responsive Design**: Optimized for everything from mobile phones up to high-resolution desktop monitors.
- **Accessibility & SEO**: Built with semantic HTML, descriptive meta tags, and accessibility best practices.
- **Security Blueprint**: Includes a professional `security.txt` configuration.

## 🛠️ Technology Stack

- **Core**: HTML5, CSS3, JavaScript (Vanilla)
- **Fonts**: JetBrains Mono, Inter (Google Fonts)
- **Icons**: Custom SVG & Unicode symbols
- **Animations**: CSS Transitions, Intersection Observer API, Glitch Effect (JS)

## 🛠️ Setup & Usage

Since this is a vanilla stack, no build steps are required.

1.  Clone the repository or download the files.
2.  Open `index.html` in any modern web browser.
3.  Enjoy the terminal experience!
